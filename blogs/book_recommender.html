Building a Personalized Book Recommender System

Published on 25-08-2025

üìö Overview

This project creates a content-based book recommendation engine that suggests similar books to readers based on their input. The focus was on Flask for the web interface and Python for backend logic.

‚öôÔ∏è Tech Stack

  • Python, Pandas, NumPy for data preprocessing
  • Pickle for saving precomputed similarity matrices
  • Flask for serving the web app
  • Bootstrap for front-end styling

üõ†Ô∏è Implementation Steps

  1. Prepared datasets with book titles, authors, images, and ratings
  2. Computed similarity using vectorized representations of books
  3. Stored processed data in popular.pkl, pt.pkl, books.pkl, and similarity_scores.pkl
  4. Created two Flask routes:
    • / ‚Üí shows popular books
    • /recommend ‚Üí search UI for user input
  5. On submit, the system fetches top 4 similar books and displays titles, authors, and cover images.

üìú Key Code Snippet

@app.route('/recommend_books', methods=['POST'])
def recommend():
    user_input = request.form.get('user_input').strip().lower()

    index_array = np.where(pt.index.str.lower() == user_input)[0]
    if len(index_array) == 0:
        return render_template('recommend.html', data=[], error="‚ö†Ô∏è Book not found!")

    index = index_array[0]
    similar_items = sorted(list(enumerate(similarity_scores[index])),
                            key=lambda x: x[1], reverse=True)[1:5]

    data = []
    for i in similar_items:
        temp_df = books[books['Book-Title'].str.lower() == pt.index[i[0]].lower()]
        if not temp_df.empty:
            data.append([temp_df['Book-Title'].values[0],
                         temp_df['Book-Author'].values[0],
                         temp_df['Image-URL-M'].values[0]])

    return render_template('recommend.html', data=data)

This route processes user input and displays the recommended books.

üí° Lessons Learned

  • Case-insensitive handling prevents mismatches with user input
  • Pickling preprocessed data speeds up the recommendation response
  • Bootstrap makes quick UI design possible without deep CSS knowledge

üöÄ Conclusion

This Book Recommender is a lightweight Flask application that shows how similarity-based approaches can deliver personalized book suggestions. Future improvements could include collaborative filtering and hybrid recommendations using user ratings.
